# Flocking Simulation with CUDA

Реализация алгоритма роевого движения (flocking) в реальном времени с использованием CUDA для вычислений и OpenGL/GLFW для визуализации.

## Возможности

- Симуляция движения агентов по трём правилам: cohesion, alignment, separation.
- Динамическое добавление препятствий и целей с помощью мыши.
- Визуализация β‑агентов (виртуальные агенты для обхода препятствий).
- Отображение связей между агентами.

## Управление

| Клавиша | Действие                              |
|---------|---------------------------------------|
| T       | Режим установки цели (клик мышью)     |
| O       | Режим добавления препятствий (клик)   |
| C       | Очистить все препятствия              |
| B       | Показать/скрыть β‑агентов             |
| G       | Показать/скрыть сетку связей          |
| X       | Убрать цель (только роевое поведение) |
| H       | Показать справку                      |
| ESC     | Выход                                 |

## Требования

- Windows 10/11
- NVIDIA GPU с Compute Capability 8.0+ (тестировалось на RTX 4080)
- CUDA Toolkit 11.0+
- CMake 3.18+
- GLFW 3.x

```bash

cd build
cmake .. -G "Visual Studio 17 2022" -A x64 -DCMAKE_TOOLCHAIN_FILE=C:\vcpkg\scripts\buildsystems\vcpkg.cmake (Свой путь)
